### *The Self-Healing Supply Chain Narrative*

**SignalChain.AI** is a predictive operational command center designed to move logistics from reactive firefighting to proactive management. By integrating risk forecasting, carbon tracking, and narrative storytelling, we empower supply chain teams to fix disruptions before they occur.

---

## üöÄ Key Features

* **Predictive Risk Scoring:** An early warning system that flags high-probability delays based on supplier reliability and route volatility.
* **"Trigger Recovery" (Digital Twin):** A one-click mitigation tool that simulates rerouting shipments and provides real-time updates to global KPIs.
* **Journey Storyteller:** Converts raw CSV data into a human-readable narrative, bridging the gap between technical logistics and customer service.
* **Sustainability Engine:** Real-time calculation of Carbon Impact ($CO_2e$) across different transport modes.
* **Intelligent Exploration:** A built-in AI assistant to query risks and delays using natural language.

## üõ†Ô∏è Technical Architecture

The project follows a decoupled full-stack architecture designed for high performance and real-time state updates.

* **Backend:** Python 3.x with **FastAPI** for high-performance asynchronous API endpoints.
* **Frontend:** Modern HTML5, CSS3 (Inter Typography), and Vanilla JavaScript.
* **Data Visualization:** **Chart.js** for animated volatility trends and risk distribution.
* **Data Processing:** **Pandas** for managing shipment datasets and calculating complex risk scores.
